## üïπÔ∏è How to Play

1. Open `index.html` in a web browser.  
2. Click **INITIALIZE LOGIN** to start the game.  
3. Incoming tasks (emails) appear as floating glass cards on the screen.  
4. Type the **exact word** shown on each email card into the input box at the bottom.  
5. Successfully completing a task:
   - Removes the email card
   - Increases your score
   - Reduces your stress level
6. Stress rises with more pending emails and over time.  
7. If stress reaches **100%**, the system crashes and the game ends.
